# esieeit-projet-si-java-groupe-5

## Présentation
Ce projet est réalisé dans le cadre du module SI à l'ESIEE IT par le groupe 5. Il s'agit d'une application Java dont le but et les fonctionnalités sont à compléter selon le cahier des charges du groupe.

## Installation
Pour installer le projet, il suffit de cloner le dépôt avec la clé SSH :

```bash
git clone git@github.com:pierre-ltr/esieeit-projet-si-java-groupe-5.git
```

## Prérequis
- Java (version recommandée : 17 ou supérieure)
- Un IDE compatible Java (ex : VS Code, IntelliJ, Eclipse)
- Git

Avant de commencer, vérifiez vos installations :

```bash
java -version
javac -version
git --version
```

## Utilisation
1. Ouvrez le projet dans votre IDE.
2. Compilez et exécutez le programme principal.
3. Suivez les instructions spécifiques du projet (à compléter selon l'avancement).
